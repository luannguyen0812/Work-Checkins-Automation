# Intern Onboarding

## What is tracked
Your daily check-in timestamp is recorded when you post "I'm online" (or similar) in the intern group. Message content beyond the check-in keyword is not stored.

## Check-in window
Monday–Friday, 09:00–17:00 EDT. You will receive a DM reminder at 13:00 if you haven't checked in.

## Privacy
Data is retained for 12 weeks then deleted. You can opt out at any time by contacting your program manager.

## Onboarding message (post in group on Day 1)
```
Hi everyone! 👋

Starting today, please check in each morning by sending "I'm online" (or "checking in") in this group.

✅ Check-in window: 09:00 – 17:00 EDT, Monday–Friday
⏰ You'll get a reminder if you haven't checked in by 13:00
📊 Attendance is tracked for the weekly program report

That's it — one message a day keeps the tracking up to date!
Questions? Message @[your handle].
```
